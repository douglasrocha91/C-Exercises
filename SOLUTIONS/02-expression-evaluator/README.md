# Expression Evaluator Solution

This is a solution to Exercise 2: Expression Evaluator with Operator Precedence.

## Features Implemented
- Parses mathematical expressions with `+`, `-`, `*`, `/`, `%`
- Handles operator precedence and associativity
- Supports parentheses for grouping
- Supports floating-point and integer numbers
- Detects division by zero
- Written in pure C11

## Building and Running

```bash
make
make run
```

## Usage
Type an expression at the prompt and press Enter. Type `exit` or `quit` to exit.
